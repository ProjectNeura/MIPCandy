ProjectNeura/MIPCandy

A Candy for Medical Image Processing

Next-generation framework for medical imaging with out-of-the-box training, inference, evaluation, and rich integrations.

Get Started Docs & Home

Key Features

Out-of-the-box Training Pipeline

Complete training workflows for segmentation and other medical tasks.

Data Handling & Formats

NIfTI/DICOM via SimpleITK, preprocessing, transforms, and visualization.

Specialized Architectures

Pre-configured models and sliding-window inference for large volumes.

Medical Metrics

Dice, IoU, precision/recall and evaluation helpers.

Experiment Tracking

Integrations with Notion, WandB, and TensorBoard.

2D/3D Visualization

From slice viewers to volume rendering (pyvista via optional extras).

Quick Start

Requirements

# Python >= 3.12
torch >= 2.0
SimpleITK
numpy, matplotlib

Install

pip install mipcandy
# Optional: for 3D visualization
pip install mipcandy[standard]
Basic Usage
import mipcandy as mc # Load medical images loader = mc.Loader() image = loader.do_load("brain_scan.nii.gz") # Create dataset dataset = mc.SupervisedDataset( images_folder="data/images", labels_folder="data/labels" ) # Train segmentation model class MyTrainer(mc.SegmentationTrainer): def build_network(self, shape): return MyUNet(shape) trainer = MyTrainer("experiments", dataloader) trainer.train(100)
